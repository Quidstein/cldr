# Keyboard imports

The XML files in this directory are importable using the keyboard syntax as follows:

```xml
<import base="cldr" path="techpreview/somefile.xml"/>
```

See [tr35-keyboard.md](../../docs/ldml/tr35-keyboards.md#Element_import)
